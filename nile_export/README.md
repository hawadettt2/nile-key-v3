# Nile Export

Digital Export Gateway for Nile Key, an Egyptian LLC specialized in supporting high-quality national exports.

## Features

- Export Shipment Management
- Supplier Management
- Importer Management
- Document Tracking
- Shipping Line Management
- Compliance Monitoring

## Installation

```bash
bench get-app https://github.com/hawadettt2/nile-key-v3
bench new-app nile_export
bench --site frontend install-app nile_export
```

## Usage

1. Navigate to "Nile Export" module
2. Create Suppliers and Importers
3. Create Export Shipments
4. Attach required documents
5. Track compliance status
