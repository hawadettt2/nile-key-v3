# Nile Export Digital Portal

Next.js public portal for supplier and importer access.

## Components (planned)

- Landing Page
- Company Profile
- Export Services
- Supplier Onboarding
- Importer Portal
- RFQ Submission
- Shipment Tracking
- Document Upload
- Compliance Status
- Contact & Support

## Tech Stack

- Next.js App Router
- TypeScript strict
- Tailwind CSS
- Radix UI
- Zod
- React Hook Form
- Frappe API

## Setup

```bash
npm install
npm run dev
```

## Deployment

See `pwd.yml` for Docker-based production deployment.
